# AI Agents & Automation Workflows

A collection of automation projects built using n8n and AI tools. These workflows handle tasks like WhatsApp orders, inventory tracking, stock reporting, chatbots, and smart email responses.

## Projects
- AI-Powered WhatsApp Order & Inventory Assistant
- Automated Daily Stock Data Fetcher & Email Reporter
- Chatbot for Course Selling Website
- Mobile Accessories Inventory System
- Smart Email Auto-Responder

## Usage
1. Clone the repo  
2. Import workflows into n8n  
3. Add your API keys  
4. Run and automate
